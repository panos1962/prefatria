Στο παρόν directory υπάρχουν files και directories ιδιαίτερα για κάθε instance
της εφαρμογής. Ορισμένα από αυτά τα αντικείμενα είναι:

	diafimisi.php		Πρόκειται για το div της διαφήμισης που εφόσον
				υπάρχει θα εμφανιστεί στο επάνω μέρος τής βασικής
				σελίδας.

	motd.php		Πρόκειται για το div του message of the day που
				εφόσον υπάρχει θα εμφανιστεί στο επάνω μέρος τής
				βασικής σελίδας κατά την επώνυμη χρήση.

	welcome.php		Πρόκειται για το div του message of the day που
				εφόσον υπάρχει θα εμφανιστεί στο επάνω μέρος τής
				σελίδας κατά την ανώνυμη χρήση. Ουσιαστικά πρόκειται
				για γενικό ενημερωτικό μήνυμα το οποίο εμφανίζεται
				στο επάνω μέρος της σελίδας υποδοχής (welcome).
				Το μήνυμα αυτό συνήθως είναι hard link στο "motd.php".

ΔΙΑΦΗΜΙΣΕΙΣ
-----------
Προκειμένου να εμφανίσουμε διαφημιστικό μήνυμα στη βασική σελίδα, αρκεί να κατασκευάσουμε file
με όνομα "diafimisi.php" στο παρόν directory ακολουθώντας το παράδειγμα "diafimisiSample.php".

ΜΗΝΥΜΑΤΑ
--------
Τα ενημερωτικά μηνύματα ημέρας (motd) αφορούν σε μηνύματα τα οποία εμφανίζονται στο επάνω μέρος
της βασικής σελίδας. Τέτοια μηνύματα μπορούμε να εμφανίσουμε και στη σελίδα υποδοχής (welcome).
Το file "motd.php" είναι το μήμυμα που εμφανίζεται στη βασική σελίδα της εφαρμογής, ενώ το file
"welcome.php" εμφανίζεται στο επάνω μέρος της σελίδας υποδοχής· συνήθως πρόκειται για hard link
στο "motd.php".
