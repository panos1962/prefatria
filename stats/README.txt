Στο παρόν directory υπάρχουν files και directories που αφορούν στα στατιστικά
στοιχεία του «Πρεφαδόρου». Τα στοιχεία αυτά δεν είναι προσβάσιμα από τον έξω
κόσμο καθώς το directory δεν βρίσκεται κάτω από το "client" directory το οποίο
δημοσιοποιείται.

Ωστόσο, στο directory "client/site" υπάρχει symbolic link "stats" το οποίο
είναι προσπελάσιμο από τον έξω κόσμο και στο οποίο αποτίθενται στατιστισικά
στοιχεία που μπορεί κάποιος να «κατεβάσει» στον υπολογιστή του για περαιτέρω
επεξεργασία. Το link «δείχνει» στο directory "stats/www".

Το πρόγραμμα "mkstats" είναι αυτό που δημιουργεί τα απαραίτητα directories
σε μια νέα εγκατάσταση.

=============================================================================@

Στο παρόν directory υπάρχει subdirectory "etos" στο οποίο φροντίζουμε να
κρατάμε πρωτογενή στοιχεία παρτίδων κατά έτος. Τα πρωτογενή στοιχεία παρτίδας
είναι εκείνα τα στοιχεία παρτίδας που λαμβάνονται υπόψη στη στατιστική
επεξεργασία. Πιο συγκεκριμένα, στο directory "etos" έχουμε files της μορφής
"YYYY.trapezi" με τους κωδικούς παρτίδων κάθε έτους, όπου "YYYY" είναι το έτος,
π.χ. για το 2014 αντιστοιχεί το αρχείο "2014.trapezi" κοκ. Φροντίζουμε επίσης
αυτά τα files να είναι ταξινομημένα με αντίστροφη αριθμητική σειρά, ήτοι από
τον μεγαλύτερο προς τον μικρότερο κωδικό παρτίδας. Αυτά τα files παράγονται
με εντολές της μορφής:

	trapezi -r -y YYYY >YYYY.trapezi

όπου "YYYY" είναι το έτος.

Στο ίδιο directory υπάρχουν files με ονόματα "YYYY.data" τα οποία περιέχουν τα
στοιχεία των παρτίδων που αναφέρονται στα αντίστοιχα files κωδικών παρτίδας.
Αυτά τα files περιλαμβάνουν τα στοιχεία παρτίδας που είναι απαραίτητα στη
στατιστική επεξεργασία και παράγονται με εντολές της μορφής:

	stats -D YYYY.trapezi >YYYY.data

Το script "trapezi.sh" μπορεί να τρέξει τις παραπάνω εντολές για τα έτη που
καθορίζουμε στο command line, π.χ.

	sh trapezi.sh 2017 2>2017.err

θα παράξει πρωτογενή στοιχεία για το έτος 2017. Καλό είναι να διαγράψουμε το
αρχείο λαθών αφού πρώτα ελέγξουμε τυχόν λάθη που παρήχθησαν από το πρόγραμμα.

-----------------------------------------------------------------------------@

Στο παρόν directory υπάρχει subdirectory "rank" στο οποίο φροντίζουμε να κρατάμε
στοιχεία που καθορίζουν την ενδεικτική βαθμολογία κάθε παίκτη. Τα στοιχεία αυτά
κρατούνται κατά έτος και προκύπτουν μαζί με τα πρωτογενή στοιχεία παρτίδας που
κρατούνται στο subdirectory "etos".

-----------------------------------------------------------------------------@

Από τα στοιχεία που υπάρχουν στο direcotry "stats/etos" παράγονται στοιχεία
που αποτίθενται στο directory "stats/www". Ένας εύκολος τρόπος να παράξουμε
αυτά τα δευτερογενή στοιχεία και να τα αποθέσουμε στο εν λόγω directory είναι
να τρέξουμε τα σχετικά shell scripts, π.χ.

	sh pektis.sh 2017
	sh stisimo.sh 2017

Καλό είναι όμως να μην τρέχουμε αυτά τα προγράμματα με το χέρι, αλλά να τα
τρέχουμε σε τακτά χρονικά διαστήματα, π.χ. κάθε 10 λεπτά της ώρας, μέσω του
cron. Για το σκοπό αυτό παρέχεται το πρόγραμμα "stats/etos.sh" το οποίο τρέχει
τα στατιστικά συγκεκριμένων ετών και εφόσον δεν καθορίσουμε έτη, τρέχει για
το τρέχον και το προηγούμενο έτος. Το πρόγραμμα "stats/etos.sh" ενημερώνει
επίσης και την ενδεικτική βαθμολογία των παικτών. Επομένως, προσθέτουμε στο
crontab την παρακάτω γραμμή:

	*/10 * * * *	sh ${PREFADOROS}/stats/etos.sh

προκειμένου να έχουμε ενημερωμένα στατιστικά στοιχεία και βαθμολογίες σε
τακτά χρονικά διαστήματα (κάθε 10 λεπτά της ώρας).
